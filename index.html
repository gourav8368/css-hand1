What is CSS?

  1. CSS stand for Cascading Style Sheets.

  2. CSS describes how HTML, elements are to be displayed on screen, paper, or in other media.

  3. CSS is not neccesaryCSS saves a lot of work. It can control the layout of multiple web page all at once.

CSS can be added to HTML documents in 3 ways:

  • Inline - by using the style attribute inside HTML elements
  • Internal - by using a style elements in the head section
  • External - by using a elements to link to an external CSS file.
  • The most common way to add CSS. is to keep to styls in external CSS files. However. in this tutorial we will use inline and internal styles,becouse this is easier to demonstrate. and easier for you to try yourself.

When comparing CSS vs ID,the defference is that CSS class applies a style to multiple elements. ID, on the other hand, applies a style to one unique element. ID is also special in that you can used a speciel URL to link directly to an element and it's used by JavaScript.

In CSS , slectors are used to target a spacific element or range of elements on a web page. Once an element has been targeted , a style or set of style can be applied to the element.

There is a wide range of selectors available. Two of the most commanly used are class and ID. Both are used to targets elements to which a style should be applied
